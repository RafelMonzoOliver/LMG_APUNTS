Layouts amb Flexbox i Grid

Casos comuns per Flexbox

  • Quan vols alinear elements en fila o columna,com botons en una barra de navegacio o imatges en una galeria
  • Quan vols distribuir l'espai entre elements en una fila
  • Quan els elements han de creixer o reduir-se per omplir l'espai disponible,com una barra lateral o una fila de cartes

Casos comuns per CSS Grid

  • Quan vols crear una pagina amb una estructura complexa
  • Quan tens un disseny en format de quadricula amb files i columnes,com un tauler de fotos o un disseny de cartes de productes
  • Quan vols controlar exactament on va cada element en la quadricula
-Quan es tracta de crear disseny flexible,adaptables i organitzats, Flexbox i CSS Grid són les dues eines més poderoses del CSS.

Flexbox avançat:Añomeacop, flex-shrink, align-self

-Flexbox és una eina molt flexible per crear disseny en fila o columna,o els elements s'alineen i ajusten automaticament segons l'espai disponible
 Flex-shrink:Controla com es redueix la mida d'un element quan no hi ha prou espai disponible
 Alignn-self:Permet que un sol element en un contenidor flex tingui una alineacio diferent a la resta 'elements'
