Variables CSS

-Les variables CSS,també conegudes com Custom Propierties,permeten definir valors reutilitzables dins d'un document CSS.Aquesta caracteristica facilita molt la gestió i manteniment de grans fulls d'estil,ja que pots centralitzar valors com colors,mides i espaiats ,i reutilitzar-los en diverses regles CSS

Definit i utilitzar variables CSS

-Les variables CSS es defineixen dins de qualsevol selector,utilitzant el prefix "--"

Beneficis de les variables en grans projectes

-L'ues de variables es especialment util en projectes grans o complexos,on necessites garantir la consistencia en colors,mides,fonts i altres estils

Variables CSS amb valors per defecte

-Pots proporcionar un valor per defecte a les variables CSS en cas de que no estiguin definides
