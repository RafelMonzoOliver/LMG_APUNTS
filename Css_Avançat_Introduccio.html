CSS Avançat:Introducció

-A mesura que els projectes web es tornen més complexos, l'us del CSS també ha d'evolucionar per oferir solucions més adaptatives,eficients i organitzades.El CSS avançat no només es centra en dissenyar una pagina web atractiva,sinó també en assegurar-se que aqeusta pàgina funcioni bé en diversos dispositius,sigui fàcil mantenir i estigui optimitzada per a un rendiment òptim

Importancia del disseny adaptable i optimitzat

-Un dels pilars del CSS avançat és el disseny responsive,que permet que les pàgines web s'adaptin a diferents ,ides de pantalla i dispositiu
  • Media Queries Avançades
  • Unitats relatives i flexibles
  • Layouts responsius amb Flexbox i Grid

Millores practiques d'organització del CSS

-A mesura que els projectes es dan més grans,és essencial mantenir una bona organitzacio del codi CSS per evitar que es torni dificil de gestionar i mantenir
  1. Modularitzacio del CSS
    • Dividir el codi CSS en fitxers més petits i tematics
    • Arquitectura CSS escalable
  2. Pre-processador CSS
    • Sass o LESS
  3. Optimitzacio del CSS
    • Minificació del CSS
    • Carregament asíncron del CSS
  4. Utilització de frameworks CSS
    • Bootstrap
    • Tailwind CSS
