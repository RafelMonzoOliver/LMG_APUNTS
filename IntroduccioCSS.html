Que es el CSS

Què és el CSS?

 -CSS és un llenguatge de fulles d'estil que s'utilitza per controlar la presentació i el disseny dels documnetss web escrits en html.CSS defineix com s'ha de visualitzar el contingut d'HTML

Importancia del CSS en el desenvolupament web

 El CSS és una peça fonamental del desenvolupament web ,ja que permet separar la presentacio visual del contingut estructural d'una pagina
  • Diseny i estètica millorats
  • Separacio de contingut i estil
  • Facillitat de manteniment
  • Millor experiència d'usuari
  • Rendiment i velocitat
