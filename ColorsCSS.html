Colors i fons en CSS

-El CSS proporciona diverses propietats per aplicar colors i fons als elements per millorar l'aspecte visual i llegibilitat de la web

Propietat colro

-La propietat color s'utilitza per canviar el color del text dins d'un element
  • Noms de colors:Utilitzant noms predefinits com red,blue,green,etc
  • Colors hexadecimals:Utilitzar el codi hexadecimal que comença en # seguit de sis car'acters
  • RGB:Utilitza valors de rgb() on cada component s'especifica entre 0 i 225
  • RGBA:Com rgb(),però amb un quart valor per l'opacitat

Propietat background-color

-La propietat s'utilitza per establir el color de fons d'un element.Igual que la propietat color,pot definir colors utilitzant noms,valors hexadecimals,rgb,etc

Propietat background

-La propietat backgroun és una manera abreujada d'especificar diverses propietats de fons alhora ,com el color,imatge de fons,repeticio,etc

Propietat background-image

-Aquesta propietat permet afegir una imarge com a fons d'un element
  • background-repeat:Controla si la imatge de fons es repeteix,els valors poden ser repeat,no-repeat,repeat-x o repeat-y
  • background-position:Defineix la posicio de la imatge de fons
  • background-sizePermet ajustar la mida de la imatge
