Selectors avançats en CSS

-Els selectors avançats en CSS permet aplicar estils amb molta més precisió i flexibilitat, permetent seleccionar elements específics en funció de la seva posicio,relació amb altres elements o atributs

Selectors d'atribut

-Els selectors d'atribut permeten aplicar estils als elements HTML que contenen un determinat atribut o valor d'atribut

Selectors combinats

-Els selectors combinats permeten seleccionar elements que tenen una relació específica amb altres elements dins del document
  • Combinador Descendent ()
  • Combinador de fills directes (>)
  • Combinador de germans adjacents(+)
  • Combinador de germans generals (~)

Selectors pseudo-classe

-Les pseudo-classes són selectors que s'utilitzen per aplicar estils a un element en un estat o condició particular.Aixo permet canviar l'estil d'un element quan l'usuari hi interactura o quan l'element compleix certs criteris  :Hover:Aplica estils quan l'usuari passa el ratolí per sobre l'element
 :nth-child():Selecciona elements en funció de la seva posició entre els seus germans
 :firts-of-type:Selecciona el primer element del seu tipus dins del seu pare.
 :not():Selecciona tots els elements excepte aquells que coincideixin amb el selector especificat

Selectors pseudo-element

-Els pseudo-elements permeten aplicar estils a una part específica d'un element,com el seu contingut abans o després, o una línia de text.
 ::before:Afegeix contingut o estils abans del contingut real d'un element
 ::after:Afegeix contingut o estils dewspres del contingut real d'un element
