Selectors en CSS

-Els selectors son part donamental del CSS perque defineixen quins elements HTML seran afectats per les regles d'estils

Selectors de tipus

-Aquest selector aplica estils a tot els elements HTML del mateix tipus

Selector de classe

Els selectors de classe permeten aplicar estils a elements HTML amb una classe especifica. S'utilitza un punt per seleccionar una classe

Selector d'id

El sector d'un id s'utilitza per aplicar estils a un únic element HTML que tingui un identificador especific.Es sol utilitzar el "#"

Selector universal

El selector universal "*" aplica estils a tots els elements d'una pagina
