Disseny bàsic amb Flexbox

-Flexbox és un sistema de disseny en CSS que proporciona una manera olt eficient de distribuir l'espai entre els elements i alinear-los dins d'un contenidor,fins i tot quan la mida dels elements és desconeguda o dinàmica

El contenidor flex

-El punt de partida per utilitzar Flexbox és establir display:flex en l'element contenidor.

Direcciò del flux

-Per defete,els elements flex es disposen en fila horitzontalment.Pots canviar la direcció amb la propietat flex-direcction

Els valors més comuns són

  • row:Els elements es col·loquen en fila
  • row-reverse:Els elements coloquen en fila però en ordre invers
  • column:Els elements es col·loquen en columna
  • column-reverse:Els elements es col·loquen en columna però en ordre invers

Alineacio horitzontal

-La propietat justify-content controla l'alineacio dles elements al llarg de l'eix principal.Pots distribuir els elements flex de diferents maneres dins del contenidor

Els valors mes comuns son

  • flex-start:Els elements s'alineen al començament del contenidor
  • flex-end:Els elements s'alineen al final del contenidor
  • center:Els elements es centre dins el contenidor
  • space-between:Es distribueixen uniformement amb espai entre ells
  • space-around:Els elements tenen espai al voltant de cadascun

Alineacio vertical

La propietat align-items controla l'alineacio dels elements al llarg de l'eix transversal.Aquesta propietat determina com s'alineen els elements flex dins del contenidor en relacio amb la seva alçada

Els valors mes comuns són

  • stretch:Els elements s'estiren per omplir el contenidor
  • flex-start:Els elements s'alineen a la part superior del contenidor
  • flex-end:Els elements sàlineen a la part inferior del contenidor
  • center:Els elements es centren dins del contenidor
  • baseline:Els elements s'alineen segons la seva linia de base de text.

Control de l'espai que ocupa cada element(flex-grow)

La propietat flex-grow especifica com els elements flex creixen dins del contenidor per ocupar l'espai disponible
