Llistes en HTML

-Es separen en 2 tipus

Llistes no ordenades(<ul>)

  • S'empra per llistes d'elements sense un ordre especific
  • Els elements es mostren amb vinyetes o altres simbols abans de cada ítem
  • Cada element de la llista es defineix amb l'etiqueta <li>

<ul>
 <li>Primer</li>
 <li>segon</li>
 <li>tercer</li>
</ul>

Llistes ordenades (<ol>)

  • S'utilitza per a llistes què l'ordre dels elements és rellevant.
  • Els elements es numeren automàticament
    • cada element de la llista també es defineix amb <li>

<ol>
 <li>Primer</li>
 <li>segon</li>
 <li>tercer</li>
</ol>
