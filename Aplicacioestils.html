Aplicació d'estils

-El css es pot aplicar a un document HTML de tres maneres principals: en línia,incrustat(dins html) i extern

Estils en linia

-Els estils en linia s'apliquen directament a un element HTML,utilitzant l'atribut style dins de la mateixa etiqueta de l'element.Aquest metode es adequat per aplicar estils a un sol element de manera rapida

Avantatges

  • Aplicacio rapida i directa a l'element

Inconvenients

  • Dificulta la mantenibilitat en llocs webs grans perque caldria aplicar l'estil a cada element manualment
  • No aprofita la capacitat de reutilitzar els estils

Estils incrustats

-Els estils incrustats es col·loquen dins de la seccio <head> d'un document HTML,dins d'una etiqueta <style>

Avantatges

  • Més eficient que els estils aa diversos elements elements d'una mateixa pagina

Inconvenients

  • Nomes afecta la pàgina on s'aplica, no es reutilitza entre diverses pagines
  • Si la pagina creix masssa,pot fer mes dificil la lectura i manteniment

Estils externs

-Els estils externs s'emmagatzemen en un fitxer CSS separat, i s'enllacen al document HTML utilitzant l'etiqueta <link> dins de la secció <head>

Inconvenients

  • Si el fitxer CSS extern no es carrega correctament ,la pagian es mostrara sense estils

Avantatges

  • Permet reutilitzar els estils en divevrses pàgines web,facilitant la consistencia i mantenibilitat
  • Facilita la separacio entre contingut i estls
  • Optimitza el temps de carrega del lloc web,ja que els navegadors poden emmagatzemar en memoria cau els fitxers CSS
