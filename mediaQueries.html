Media Queries(Introducció)

-Les Media Queries són una característica poderosa del CSS que permet adaptar el disseny d'una pagina web en funció de les característiques del dispositiu que l'està visualitzant,com la mida de la pantalla,la resolució o l'orientacio

Què són les Media Queries?

-Un media query es una confició que s'afegeix al CSS per aplicar regles d'estil només quan es compleixin cartes característiques del dispositiu del dispositiu o de la finestra del navegador

Aplicació bàsica de Media Queries per a pantalles de diferent mida

-Un ús típic de les media queries és adaptar el disseny a diferents amplades de pantalla

Tipus de condicions més comunes en les Media Queries

  • Amplada de pantalla
  • altura de la pantalla
  • Orientació de la pantalla

Ús de múltiples condicions en una Media Query

Pots combinar diverses condicions en una sola media query mitjançant les paraules clau o only per especificar estils molt concret
