Etiquetes

Etiqueta <center>

 -L'etiqueta <center> s'utilitzava en versions antigues d'HTML per centrar el contingut dins d'una pàgina web,tant text,imatges,etc
 -Actualment esta obsoleta en HTML5,preferinse el CSS per centrar elements de manera eficient

Etiqueta<span>

 -L'etiqueta <span> es una etiqueta inline que serveix per estilitzar o manipular petites proporcions de text o contingut dins d'una pagina
 -Tambe és una etiqueta no semàntica emprada princppalment per aplicar estils o funcionalitat a un grup específic d'elements dins d'un text

Etiqueta <hr>

 -En HTML5, l'ús de l'etiqueta <hr> no és només decoratiu,Té un valor semàntic per indicar un canvi de tema o una ruptura en el flux del document,
 ajudant tant als motors de cerca com a les eines d'accessibilitat a entendre millor l'estructura del contingut

Etiqueta <template>

 -L'etiqueta <template> es utilitzada per definir fragments d'HTML reutilitzables que no es mostren immediatament quan la pàgina es carrega.El contingut en <template> no s'afegeix al document fins que es manipula explícitament amb Javascript
 -Aquest element és ideal quan vols tenir estructures html preparades que podràs afegir al document en temps real.

Etiqueta <Iframes>

 -<iframe> és una etiqueta que permet inserir una pàgina html dins d'una altra pàgina.És una manera més moderna i segura d'incrustar contingut extern
 -Els iframess'utilitza sovin per a continguts dinàmics com vídeos de youtube,google maps,etc
 L'etiqueta <iframe> es defineix fins del cos del document i es pot personalitzar amb atribtus com width, height,src i frmeborder
