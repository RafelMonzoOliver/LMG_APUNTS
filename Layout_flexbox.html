Layouts amb Flexbox i Grid

Casos comuns per Flexbox

  • Quan vols alinear elements en fila o columna,com botons en una barra de navegacio o imatges en una galeria
  • Quan vols ditribuit l'espai entre elements en una fila
  • Quan els elements an de creixer o reduir-se per omplir l'espai disponible,com en una barra lateral o una fila de cartes

Casos comuns per CSS Grid

  • Quan vols crear una pàgina amb una estructura complexa
  • Quan tens un disseny en format de quadricula amb files o columnes,com un tauler de fotos o un disseny de cartes de productes
  • Quan vols controlar exactament on va cada element en la quadricula

Flexbox avançat:Alineació,flex-shrink,align-self

Flexbox es una eina molt flexible per crear dissenys en fila o columna,on els elements s'alineen i ajusten automàticament segons lèspai disponible
flex-shrink:Controla com es redueix la mida d'un element quan no hi ha prou espi disponible
align-self:Permet que un sol element en un contenidor flex tingui una alineacio diferent a la resta d'elements.Els valors poden ser flex-start,flex-end,center,baseline o stretch

Grid Layout:creacio de quadricules

-CSS Grid Layout és una eina més potent que Flexbox quan es tracta de crear layouts en dues dimensions.Grid permet col·locar elements en una quadricula amb molta precisió, controlant la mida de les files i columnes,i permetent que els elements s'ajustin a l'espai disponible

Propietat clau en CSS Grid

-grid-template-columns i grid-template-rows:Defineix el nombre i la mida de les columnes i files dins del contenidor grid
-grid-columns i grid-row:Permeten que un element ocupi diverses columnes o files dins de la quadricula
-gap:Defineix l'espai entre les files i columnes de la quadricula

Ús combinat de Flexbox i Grid

-Tot i que Flexbox i Grid es poden utilitzar per separat,sovint es util combinar-los per obtenir el millor de tots dos
