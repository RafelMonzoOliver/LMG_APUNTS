Dimensions i caixes en CSS

-El CSS s'utilitza el model de caixa per representar els elements d'una pagina web.Cada element es considera com una caixa rectangular que conte el contingt,farcits,marges,vores,etc
  • Contingut:El contingut real de l'element,com text,imatges o altres elements
  • Farcit:L'espai entre el contingut i la vora
  • Vora:El limit al volant del farcit
  • Marge:L'espai extern entre la vora i altres elements

Propietas de dimensions

-Les propietats width i height s'utilitzen per definir la mida d'un element.Aquest valor pot ser definit en una unitat com pixel o percentatges.

El model de caixa en CSS

El model de caixa estableix com es calculen les dimensions d'un element ,tenint en compte les propietats de marge,farcits i vores.

Propietat box-sizing

  • content-box:Nomes inclou el contingut en les dimensions,mentre que les vores i els farcits s'afegeixen a part
  • border-box:Inclou el contingut,el farcit i la vora dins de les dimensions especificades

Control de desbordament

-Quan el contingut d'un element es mes gran que les dimensions especificades,el CSS ofereix la propietat overflow per controlar com es mostra el contingut que desborda

Els valors mes comuns

  • visible:El contingut que desborda es mostra fora de l'element
  • hidden:El contingut que desborda es retalla i no es mostra
  • scroll:Mostra barres de desplaçament encara que el contingut no desbordi
    • 7
  • auto:Mostra barres de desplaçament nomes si el contingut desborda
